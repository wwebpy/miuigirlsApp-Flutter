## Getting Started

### Prerequisites
- Flutter SDK 3.0.0 or higher
- Xcode (for iOS development)
- iOS Simulator or physical iOS device

### Installation

1. Clone the repository
2. Install dependencies:
```bash
flutter pub get
```

3. Generate Hive adapters:
```bash
flutter packages pub run build_runner build
```

4. Run the app:
```bash
flutter run
```
